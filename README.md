# Rekomendasi_Makanan_Berdasarkan_budget_dan_bahan
Aplikasi Python terminal yang menampilkan dan mencari menu makanan Jawa dari file CSV. Mendukung pencarian berdasarkan bahan atau budget, menampilkan detail lengkap seperti bahan, nutrisi, dan instruksi memasak. Cocok untuk belajar pengolahan data CSV dan pembuatan menu interaktif.
